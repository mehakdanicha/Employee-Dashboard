Employee Management Dashboard

A responsive and professional dashboard built with React.js and Firebase, featuring a sidebar, dashboard cards, notifications, and full CRUD operations for employees.

Features

Sidebar navigation with hover effect

Dashboard cards showing statistics

Recent Activity & Notifications section

Employee CRUD (Add, Edit, Delete) using Firebase

Professional employee table with hover effect

Responsive design for desktop and mobile

Tech Stack

Frontend: React.js, CSS3

Backend: Firebase Firestore

Routing: React Router DOM

Icons: React Icons
